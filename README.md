### Technologies used to build ALE-gorithm:
```
- App Back-End: Python Flask | SQLAlchemy
- Database: MySQL | Google Cloud Platform (GCP) 
- Data Visualization: Javascript | Plotly.js | D3.js | AnyChart.js
- Front-End: HTML | Bootstrap | CSS
- Testing: Postman
```
